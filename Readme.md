# 🌿 R&B Pine Straw Landscaping Website

A clean, modern landing page built for a landscaping business focused on **pine straw installation** and additional outdoor services.

This project is designed to be **simple, fast, and conversion-focused**, prioritizing clarity, trust, and real work visuals over unnecessary complexity.

---

## 🚀 Features

* Responsive navbar with mobile menu (Lucide icons)
* Full-screen hero with image carousel
* Smooth transitions and subtle animations
* Clean typography (Cinzel + Inter)
* Glass-style navbar with blur effect
* CTA-focused layout for better conversions

---

## 🧱 Tech Stack

* **HTML5**
* **CSS3 (Custom Properties / Variables)**
* **Vanilla JavaScript**
* **Lucide Icons**

No frameworks. No unnecessary dependencies. Just clean, maintainable code.

## 📱 Responsive Design

The layout adapts to different screen sizes:

* Desktop → full navbar + CTA
* Tablet → adjusted spacing
* Mobile → hamburger menu + floating panel

---

## 🎨 Design System

### Colors (CSS Variables)

* Primary: Green tones (brand action)
* Secondary: Earth tones (support)
* Background: Neutral light tones
* Text: Dark neutral for readability

### Typography

* **Cinzel** → headings / brand feel
* **Inter** → body text / UI clarity

## ⚙️ Setup

1. Clone or download the project
2. Place your images inside `/images`
3. Update hero images in CSS or HTML
4. Open `index.html` in your browser

No build step required.
